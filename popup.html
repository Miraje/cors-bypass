CORS Bypass

Enabled

This extension uses Chrome Debugger API for the active tab only:

Adds Origin header to requests:
Origin: [request URL origin]

Example:
Request to https://api.com adds
Origin: https://api.com

⚠️ You'll see "DevTools is debugging" message when active

⚠️ Use only for development purposes. Do not use in production.

⚠️ Disable it when not needed.
